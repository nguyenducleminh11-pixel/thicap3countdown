# 🎯 Thi Cấp 3 Countdown

Một trang web **đếm ngược đến ngày thi cấp 3** giúp giữ động lực học tập mỗi ngày.
Mỗi khi mở tab lên là thấy thời gian còn lại → **không dám lười nữa**.

## 🚀 Demo

👉 https://nguyenducleminh11-pixel.github.io/thicap3countdown/countdown.html

## ✨ Tính năng

* ⏳ Đếm ngược **ngày / giờ / phút / giây** đến ngày thi
* 🔄 Cập nhật **real-time mỗi giây**
* 💡 Hiển thị **motivation quotes**
* 🌐 Host bằng **GitHub Pages**
* ⚡ Trang nhẹ, mở rất nhanh

## 🛠️ Công nghệ sử dụng

* **HTML**
* **CSS**
* **JavaScript**

Không dùng framework → đơn giản, dễ chỉnh sửa.

## 🎯 Mục tiêu

Tạo một trang nhỏ để:

* Nhắc bản thân **thời gian không còn nhiều**
* Tăng **động lực học mỗi ngày**
* Luyện tập **web development cơ bản**
